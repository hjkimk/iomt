Final Submission Files for Project Internet of Medical Things

To reproduce the results:

1. Data Preprocessing - Process the logged data in the data_collection folder using the read_file_bluesee_1 python script.

2. Algorithm Implementation - Create the dataframe for model inputs, visualize data, and train classification models SVM and Decision Tree using the classification_algorithm_2 python script.

3. Demo - Visualize unseen data in the demo folder in real-time using the demo_3 python script.

Additional information:
- The models used for the results is saved in the trained_ML_models folder.
- The real-time demo visualization videos are saved in the demo_result_animations folder.
